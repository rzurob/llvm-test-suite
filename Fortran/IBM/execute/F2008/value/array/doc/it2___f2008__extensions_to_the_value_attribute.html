All the functional tests specified below will test that the dummy argument has the same value and properties as the effective argument.

The value of the dummy argument within the procedure should be the same as the effective argument.

Changes made to a dummy argument with a value attribute should not affect the actual argument. 

Most of the tests will be done with an array of size 10, rank 1 unless otherwise specified.
