Diagnostic:

Scenario Description Results
intrinsic01D.scenario Test the new math intrinsic procedures ACOSH, ASINH, ATANH, ATAN and LOG_GAMMA with different arguments:
   integer
   character
   negative integer
   zero
   string
   logical
   real
   complex
   ATAN: both arguments zero, complex and different types		 Verify the error messages in the output file
intrinsic02D.scenario Test the new and extended math intrinsic procedures ACOS, ASIN, ATAN, COSH, SINH, TAN, TANH, ATAN, GAMMA, ACOSH, ASINH, ATANH and LOG_GAMMA with different levels of the language:
   77
   90
   95
   2003
   2008

 Verify the error messages in the output file. Any level below 2008 should return a warning message 1518-256 (L) mentioned in IT1 [1]. That's the only message checked in the output.
