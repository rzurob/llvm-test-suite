Overview


Fortran 2008 standard adds a new function into the ISO_C_BINDING module, C_SIZEOF, which is similar to the sizeof operator in C.

The syntax for the function is

15.2.3.7 C SIZEOF (X)
- Description. Size of X in bytes.
- Class. Inquiry function.
- Argument. X shall be an interoperable data entity that is not an assumed-size array.
- Result Characteristics. Scalar integer of kind C_SIZE_T (15.3.2).
- Result Value. If X is scalar, the result value is the value that the companion processor returns as the result of applying the sizeof operator to an object of a type that interoperates with the type and type parameters of X.
- If X is an array, the result value is the value that the companion processor returns as the result of applying the sizeof operator to an object of a type that interoperates with the type and type parameters of X, multiplied by the number of elements in X.

Test Strategy


The feature includes the functional test and diagnostic test.


The functional test will cover the following :


- Test X is an interoperable intrinsic type, including

  • - type INTEGER with the kind type parameter in the following table

C_INT
C_SHORT
C_LONG
C_LONG_LONG
C_SIGNED_CHAR
C_SIZE_T
C_INT8_T
C_INT16_T
C_INT32_T
C_INT64_T
C_INT_LEAST8_T
C_INT_LEAST16_T
C_INT_LEAST32_T
C_INT_LEAST64_T
C_INT_FAST8_T
C_INT_FAST16_T
C_INT_FAST32_T
C_INT_FAST64_T
C_INTMAX_T
C_INTPTR_T

  • - TYPE REAL with the kind type parameter in the following table
C_FLOAT
C_DOUBLE
C_LONG_DOUBLE

  • - TYPE COMPLEX with the kind type parameter in the following table
C_FLOAT_COMPLEX
C_DOUBLE_COMPLEX
C_LONG_DOUBLE_COMPLEX


  • - TYPE LOGICAL with the kind type parameter C_BOOL

  • - TYPE CHARACTER. with the kind type parameter C_CHAR

- Test X is type(C_PTR) which is interoperable with C pointer type

- Test X is type(C_FUNPTR) which is interoperable with C function pointer type

- Test X is derived types that is interoperable with a C struct type

- Test X is expression and function reference

- Test X is constant

- Test X is global variables declared in module

- Test X is array or scalar of interoperable intrinsic types and interoperable derived-types

- Test if result type of C_SIZEOF is integer(C_SIZE_T)

- Apply -qalign option to Fortran and C code if there is a misalignment for a derived-type, including 7 bindc suboptions power,bit_packed,full,mac68k, natural, packed and twobyte.

- Apply -qlongdouble to C program when type is real(C_long_double) or complex(c_long_double_complex)

- Test both -q64 and -q32 options.

The diagnostic test cases check if the compiler will flag messages if X is

- an assumed-size array

- not interoperable type
