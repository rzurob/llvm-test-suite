Overview

According to Fortran 2008 standard, the coindexed object is the data object whose designator includes an image-selector which determines the image index.

A procedure is either a function or a subroutine.


Test Strategy

The related procedure call statement behavior with coarray will be tested separately.
Tested here coverage will be only procedure call statement with different corank using for coindexed object which is intrinsic type. Derived types aren't included this time.
Functional:
Test that procedure call statements correctly work in the program, including internal, external, module procedures and procedures pointers. (function and subroutine)
