Overview

According to Fortran 2008 standard, the coindexed object is the data object whose designator includes an image-selector which determines the image index.

Intrinsic assignment: variable = expr

Test Strategy

The related coindexed object assignment statement behavior with coarray will be tested separately.
Tested here coverage will be only assignment statement using for coindexed object which is intrinsic type. Derived types aren't included this time.


Functional:
Test that assignment statements correctly work in the program, such as intrinsic and masked array assignments.

All scenario should include A[p] = B[q], A[p] = local1, local2 = B[q]

Diagnostic:
Negative: compile-time error message issued when coindexed object with incorrect code.
