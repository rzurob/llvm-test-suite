IT2 Comments

The character 'x' will be added to some scenario file names to signify 1 or more testcases will be created for that particular item.

num_images()

DIAGNOSTIC

Scenario Name Description Expected Results
num_images_d001a.scenario Invalid Arguments Testing. Does not accept any arguments. Verify compiler error message.
num_images_d001b.scenario Invalid Arguments Testing. Test num_images() intrinsic calls with variable declarations and statements of the same name.
 Verify compiler error message.
num_images_d002.scenario Return Type Testing. Returns default integer, other incompatible types will report an error. Verify compiler error message.
num_images_d003.scenario CAF Unsupported. Call num_images() with CAF off via -qnocaf. Verify compiler error message.


FUNCTIONAL

Scenario Name Description Expected Results
num_images_f001.scenario Valid Values #1. Num_images() produces a value >= 1. Verify result >= 1.
num_images_f002.scenario Correct Values #2. Num_images() produces the value n when -qcaf=images=n is set. Verify result matches the option set value.
num_images_f003.scenario Consistent Results. Num_images() produces consistent results throughout a program. Verify num_images() result persists.


this_image()

DIAGNOSTIC

Scenario Name Description Expected Results
this_image_d001a.scenario Invalid Arguments Testing #1a. Test calls to this_image() intrinsic mixed with variable declarations and statements of the same name. Verify compiler error message.
this_image_d001b.scenario Invalid Arguments Testing #1b. Accepts only 0 to 2 arguments. Anything else is an error. Verify compiler error message.
this_image_d002.scenario Invalid Arguments Testing #2. First argument must be of type Coarray. Test Non-coarray and coindexed args. Verify compiler error message.
this_image_d003.scenario Invalid Arguments Testing #3. Second argument must be of type integer scalar. Test non integer types for the 2nd arg with a coarray as the 1st arg. Verify compiler error message.
this_image_d004.scenario Return Type Testing #1. Provide 0 or 2 args to this_image(). This returns a default integer scalar, other incompatible types will report an error. Verify compiler error message.
this_image_d005.scenario Return Type Testing #2. Provide 1 arg to this_image(). This returns a default integer array of rank=1. Verify compiler error message.
this_image_d006a-b.scenario Index Out of Bounds. Provide an integer for the 2nd arg of this_image that is outside the range of 1 to n where n is the corank of the 1st arg.
 Runtime error.
this_image_d007.scenario CAF Unsupported. Call this_image() with CAF off via -qnocaf. Verify compiler error message.


FUNCTIONAL

Scenario Name Description Expected Results
this_image_f001a-c.scenario Valid Values. All instances of this_image() produces a value >= 1. Test a) this_image(), b) this_image(caf), and c) this_image(caf, n).
 Verify result >= 1.
this_image_f002a-c.scenario Valid Values. All instances of this_image() produces a result <= num_images() for all images. Test a) this_image(), b) this_image(caf), and c) this_image(caf, n). Verify correct output.
this_image_f003.scenario Check Result. This_image(z) and this_image(z, n) results must satisfy any declared cobounds of coarray z. Verify correct output.
this_image_f004.scenario Check Result. Image_index(z, this_image(z)) == this_image(). Verify correct output.
this_image_f005.scenario Check Result. This_image(z, n) produces the same result as the nth item in this_image(z) array for all n. Verify correct output.
this_image_f006a-b.scenario Consistent Results. This_image() produces consistent results throughout a program for a given image. Verify this_image()
has consistent output.
