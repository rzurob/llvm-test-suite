Overview


Test Strategy

The test plan for the image_index() CAF intrinsic will be based on the F2008 standard.

  • -qcaf will be a required xopt for all scenarios unless otherwise stated.
  • -qcaf=images and -qcaf=dnodes will only be specified for testcases where the image count has a direct impact on the testcases' results.
  • Testcases will be designed to be platform independent.
Diagnostic testing will consist of argument and return type testing.

 Functional testing will consist of testing image_index() with different parameter values for different image sizes to see that the results are produced correctly and consistently.

Test Procedures/Scenarios

The character 'x' will be added to some scenario file names to signify 1 or more testcases will be created for that particular item.
