IT2 Comments

Diagnostic

Scenario Name Description Expected Results
image_index_d001.scenario Invalid Arguments Testing #1. First argument must be a coarray object of any type. Anything else should flag an error.
 Verify compiler error message.
image_index_d002.scenario Invalid Arguments Testing #2. Second argument must be a rank=1 integer array.
 Verify compiler error message.
image_index_d003.scenario Return Type Testing. Returns a default integer scalar. Other incompatible types will flag an error. Verify compiler error message.
image_index_d004.scenario CAF Unsupported. Call image_index() with CAF off via -qnocaf. Verify compiler error message.
image_index_d005.scenario Invalid corank. Second argument must have the same number of elements as the coarray's corank. Verify compiler error message.
image_index_d006a.scenario Number of Arguments. Test image_index() calls with an invalid number of arguments.
 Verify compiler error message.
image_index_d006b.scenario Number of Arguments. Test calls to image_index() intrinsic mixed with variable declarations of the same name.
 Verify compiler error message.

Functional

Scenario Name Description Expected Results
image_index_f001a.scenario Invalid Sequence of Cosubscripts. The 2nd argument is an invalid sequence based on the corank. Test with 1 image.
 Result = 0.
image_index_f001b.scenario Invalid Sequence of Cosubscripts. The 2nd argument is an invalid sequence based on the corank. Test with multiple images.
 Result = 0.
image_index_f002.scenario Valid Sequence. The image value returned must be >= 1 and <= num_images().
 Result >= 1 && Result <= num_images()
image_index_f003.scenario Verify results of image_index with corank=1 coarrays on all images. Result should match value in array.
image_index_f004.scenario Verify results of image_index with corank=2 coarrays on all images. Result should match coarray structure.
image_index_f005.scenario Verify results of image_index with multi dimensional corank coarrays on all images. Result should match coarray structure.
image_index_f006.scenario Verify results of image_index with this_image(caf) as the 2nd argument.
 Result should match this_image() result.
image_index_f007.scenario Test image_index with use associated coarrays in the first argument. Result should match coarray structure.
image_index_f008.scenario Test image_index with deferred shape arrays in the second argument. Result should match coarray structure.
