Overview

%Note{}%

There is no I0 for this feature. This document describes testing one of intrinsic types Character in a CAF (Coarray Fortran, which has now become part of the Fortran 2008 standard) environment. Although there are some existing test cases using character, but it's necessary to add coverage to character coarrays that otherwise will be missed by general coverage. For detail description of this intrinsic type refers to the Fortran 2008 standard [1]. For detail introduction on CAF read [2].

Test Strategy

%Note{Test strategy used in the test}%

The feature only need functional testing.


Following are some of the considerations while writing the new test cases:
    (1) test different lengths of characters, including zero-length character.
    (2) test character coarray variables in modules as global variables.
    (3) test character array coarray, also test different dimensions, co-rank and co-bound for the coarray.
    (4) test character component of derived type coarray (without attribute allocatable and pointer now)
    (5) test these two cases for substring, dummy args, concatenation, and file I/O for
         a.) coarray object (i.e. a = b // c)
         b.) coindexed object (i.e. a[1] = b[1] // c[2])

Functional Test Points:

1.1 Concatenation
    Test whether the concatenation of several strings on same or different images can be executed correctly and the correctness of the results need to be checked.
   
1.2 Substring
    Test whether the substring of several strings on same or different images can be executed correctly and the correctness of the results need to be checked.
   
1.3 Dummy arguments   
    Test the correctness of calling procedures which use local or other images' strings as their dummy arguments.

1.4 File I/O (internal file)   
    Test whether reading from STD_INPUT and writing to STD_OUTPUT by some character type variables on local or other images can be executed correctly and the correctness of the results need to be checked.

1.5 File I/O (external file)
    Test whether reading and writing to a file with character data on local or other images can be executed correctly and the correctness of the results need to be checked.  (maybe just 1 example run on 1 image since multiple image is not yet supported).

 

Test Procedures/Scenarios

%Note{List and brief descriptions of the test cases created for the feature}%


Scenario Name
Description
1.1 Concatenation
 
for all concatenation cases All concatenation cases will cover test points below:
(1) normal-concatenation:
   coarray                // local
   coarray[me]        // local
   coarray[P]           // local
   coarray1             // coarray2[P]
   coarray1[me]     // coarray2[P]
   coarray1             // coarray2
   coarray1[me]     // coarray2[me]
   coarray1             // coarray2[me]
(2) self-concatenation:
   coarray               // coarray
   coarray               // coarray[me]
   coarray[me]       // coarray[me]
(3)  the concatenation between equal or more than three objects
       the objects would consist of coarray, coindexed, local variables
(4) image P will concatenate one coarray from each of other images (considering the time, the max is 16)
       if(me == P) then ...
                  do i = 1, ne
                      concateStr = trim(concateStr) // coStr1[i]     ......
                  end do         
(5) image Q will modify the value of some coarray on image P like:
          if(me == Q) then
                coConcateStr[P] = coStr1[P]//coStr1[Q]
                ...
          end if
(6) will do the concatenation on any two images like:
    do i = 1, ne
        coConcateStr = coStr2//coStr2[i]
        ....
    end do
concate001.scenario
Test the concatenation of zero-length character coarray
(1) cover all the test points for all concatenation cases
(2) test whether the initialization is correct when the assign value has more than zero length
(3) test zero-length concatenate zero-length will get zero-length
(4) test zero length concatenate n-length will get n-length
concate002.scenario
Test the concatenation of different lengths of character coarrays
(1) cover all the test points for all concatenation cases
(2) test whether both the length and the content on the result of concatenation is correct
concate003.scenario Test the concatenation of character coarray variables in modules
(1) cover all the test points for all concatenation cases
(2) similar to concate002 but mainly consider the different length of coarray variables in module, also local variables in module
concate004.scenario Test the concatenation of character array coarray with different codimensions
(1) cover all the test points for all concatenation cases
(2) similar to concate002 but mainly consider array coarray, also consider the codimensions.
(3) use the character in array coarray to initialize other characters (using repeat)
(4) test two and three co-dimension of character array coarray.
concate005.scenario Test the concatenation of character component of derived type coarray
(1) cover all the test points for all concatenation cases
(2) similar to concate002 but mainly consider the derived type coarray which has character component.
   
1.2 Substring
 
for all substring scenarios All concatenation cases will cover test points below:
(1) verify the length and the content is correct for the combination for a) and b)
     a.) will cover coarray in form coarray, coarray[me] and coarray[P]
     b.) the start pos and the end pos for substring:
          - from the beginning to the end
          - the first character (length one)
          - the last character (length one)
          - the middle character (length one)
          - the middle characters (length more than one)
          - some substring including the first character
          - some substring including the last character
(4) image P will substring from each of other images, then assign to some interval of one substring, then compare the string with another string by concatenation.
    if(me == P) then
        do i = 1, ne
            coStr4 = trim(coStr4) // coStr1(1:5)[i]
        end do
        do i = 1, ne
           loStr((i-1)*5+1:i*5) = coStr1(6:10)[i]
        end do
     ....
(5) image Q will modify the value of some coarray on image P, similar to (4)
(6) each image substring from each of the others, then verify the results
substr001.scenario  
Test the substring of different lengths of character coarray
(1) cover all the test points for all substring cases
(2) the length would include 10, 100 and 1000.
substr002.scenario
Test the substring of character coarray variables in modules
(1) cover all the test points for all substring cases
(2) similar to substr001, mainly consider the different length of coarray variables in module, also local variables in module
substr003.scenario Test the substring of character array coarray with different codimensions
(1) cover all the test points for all substring cases
(2) similar to substr001, mainly consider array coarray, also consider the codimensions.
substr004.scenario Test the substring of character component of derived type coarray
(1) cover all the test points for all substring cases
(2) similar to substr001, mainly consider character component of derived type coarray
   
1.3 Dummy arguments
 
dummyArg001.scenario
test dummy arg character coarray
(1) test subroutine and function in program with one coarray and two coarray character dummy arguments
(2) test subroutine and function in module with two character coarray dummy arguments
(3) test external subroutine and function with two character coarray dummy arguments
(4) will combine with substring and concatenation
dummyArg002.scenario
test dummy arg character array coarray
(1) test one-dim, two-dim, three-dim character array coarray with one-dim, two-dim, three-dim co-dim
dummyArg003.scenario test dummy arg character array coarray with different corank and cobound from the caller
(1) similar to dummyArg002 but use different corank and cobound
   
1.4 File I/O (without file)
 
charIO001.scenario
test that reading character coarray from STDINPUT and writing character coarray to STDOUTPUT
(1) test different images read different length characters from stdinput
(2) write to stdoutput, then verify the result
   
1.5 File I/O (with file)
 
charIO002.scenario
test that reading and writing character coarray to one file
(1) test different images open/read/write/close the same file, then verify the result of read and write.
