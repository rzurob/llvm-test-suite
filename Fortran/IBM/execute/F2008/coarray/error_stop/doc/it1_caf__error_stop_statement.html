Overview

%Note{Simple description of the feature; In which product it is first implemented, is it common for other platforms, is it common for both C and C++ and so on.}%


FORTRAN F2008 standard introduces a new statement, ERROR STOP, the execution of which initiates an error termination of execution of all images.

Syntax:
  R856  error-stop-stmt  is  ERROR STOP [ stop-code ]
  R857  stop-code          is  scalar-default-char-constant-expr
                                     or  scalar-int-constant-expr
C850  (R857) The scalar-int-constant-expr shall be of default kind.

For the feature’s implementation details, see the I0 document.

Test Strategy

%Note{Test strategy used in the test}%

   The related STOP statement behavior with coarray will be tested separately. Anyway, STOP statement could be also included in some of this feature test cases.
   Tested here coverage will be only ERROR STOP statement use for coarrays. The extended test for non-CAF use will be added in the next deliverable.

   In general the testing will be divided into two parts: Diagnostic and Functional Testing.
   Diagnostic:
       Positive:  looks for the expected run-time messages produced by STOP RUN, change in the program exit status, with and without errimagenum use.
       Negative: compile-time error message issued when ERROR STOP with incorrect stop-code or in incorrect place
   Functional:
      Test that ERROR STOP correctly terminates the work on the program.
